# Mecanic Manager v1.0

Sistema integral de gestión para talleres mecánicos, lubricadoras y ventas de repuestos.

## Características Principales
- ✅ Gestión de Inventario con QR
- ✅ Sistema POS (Punto de Venta)
- ✅ Gestión de Servicios Mecánicos
- ✅ CRM - Gestión de Clientes
- ✅ Control de Créditos
- ✅ Gestión de Empleados y Comisiones
- ✅ Notificaciones Inteligentes  
- ✅ Integración WhatsApp
- ✅ Reportes y Estadísticas

## Instalación Rápida

1. Instalar dependencias:
```bash
npm install
```

2. Configurar Supabase:
- Crear proyecto en https://supabase.com
- Copiar credenciales a `config/supabase.js`
- Ejecutar `database/schema.sql` en Supabase

3. Iniciar app:
```bash
npm start
```

Ver **INSTALACION.md** para instrucciones detalladas.

## Stack Tecnológico
- React Native + Expo
- Supabase (PostgreSQL)
- Twilio (WhatsApp - opcional)
